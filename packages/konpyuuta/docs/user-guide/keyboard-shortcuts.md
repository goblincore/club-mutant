# Keyboard Shortcuts Reference

Complete keyboard shortcuts for Debian Time Capsule.

## Global Shortcuts

### Window Management

| Shortcut    | Action                  |
| ----------- | ----------------------- |
| `Alt+F4`    | Close active window     |
| `Alt+F9`    | Minimize window         |
| `Alt+F10`   | Maximize/restore window |
| `Alt+Tab`   | Switch between windows  |
| `Alt+Space` | Open window menu        |

## Tips for Learning Shortcuts

### Start with These 5

1. `Ctrl+Alt+1-4` - Switch workspaces
2. `Alt+F4` - Close window
3. `Ctrl+Alt+E` - Open XEmacs
4. `Ctrl+X Ctrl+S` - Save in XEmacs
5. `Alt+Tab` - Switch windows

### Practice Daily

- Use shortcuts instead of mouse for one week
- Focus on one category at a time
- Print this reference and keep it nearby

### Muscle Memory

After 2-3 weeks of consistent use, shortcuts become automatic.

## Customization

Want to change shortcuts? Edit your settings in Style Manager → Keyboard section.

## Conflicts with Browser

Some shortcuts may conflict with your browser:

- `Ctrl+W` - Browser closes tab, XEmacs cuts text
- `Ctrl+T` - Browser opens new tab

**Solution:** Use CDE as a PWA (installed app) to avoid conflicts.

## Platform-Specific Notes

### Windows

- `Alt` key works as expected
- `Ctrl` key works as expected

### macOS

- Use `Cmd` instead of `Ctrl` for some shortcuts
- `Option` works as `Alt`

### Linux

- All shortcuts work as documented
- May conflict with desktop environment shortcuts
