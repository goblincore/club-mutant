# Style Manager User Guide

Customize your CDE desktop with 76 authentic color palettes and 168 original backdrops.

## Opening Style Manager

- Click the Style Manager icon (palette) in the Front Panel
- Use keyboard shortcut: `Ctrl+Alt+S`
- Open from App Manager

## Sections

### Color

Choose from 76 authentic Motif color palettes:

**Popular Palettes:**

- **Platinum** - Classic gray (default)
- **Broica** - Warm browns
- **Alpine** - Cool blues
- **Midnight** - Dark theme
- **Coalmine** - Very dark
- **Camouflage** - Earth tones

**How to Apply:**

1. Click "Color" section
2. Browse palette list
3. Click a palette to preview
4. Click "Apply" to use it

### Backdrop

Choose from 168 original XPM backdrops:

**Categories:**

- **Patterns** - Geometric designs
- **Textures** - Natural textures
- **Abstract** - Artistic patterns
- **Tech** - Circuit boards, grids

**Popular Backdrops:**

- Afternoon
- CircuitBoards
- BrokenIce
- Carpet
- Canvas

**How to Apply:**

1. Click "Backdrop" section
2. Browse backdrop list
3. Click to preview
4. Click "Apply" to use it

### Font

Adjust text size for better readability:

**Sizes:**

- Small (12px)
- Medium (14px) - Default
- Large (16px)
- Extra Large (18px)

**How to Change:**

1. Click "Font" section
2. Select size
3. Click "Apply"

### Keyboard

Customize keyboard behavior:

- **Key repeat rate** - How fast keys repeat
- **Key repeat delay** - Delay before repeat starts
- **Beep on error** - Sound feedback

### Mouse

Adjust mouse settings:

- **Double-click speed** - Fast, Medium, Slow
- **Pointer acceleration** - Mouse sensitivity
- **Left/Right handed** - Swap buttons

### Beep

Configure system sounds:

- **Volume** - Sound level
- **Tone** - Beep frequency
- **Duration** - Beep length

### Screen

Display settings:

- **Screen saver** - Enable/disable
- **Timeout** - Minutes until activation
- **Lock screen** - Require password

### Startup

Choose what happens at startup:

- **Resume session** - Restore windows
- **Home session** - Start fresh
- **Current session** - Save current state

### Window

Window behavior:

- **Focus follows mouse** - Auto-focus windows
- **Click to focus** - Click to activate
- **Raise on focus** - Bring to front

## Creating Your Perfect Setup

### Step 1: Choose a Palette

Browse all 76 palettes and find one you like. Consider:

- **Light themes** - Better for daytime
- **Dark themes** - Easier on eyes at night
- **Colorful themes** - More personality

### Step 2: Match a Backdrop

Choose a backdrop that complements your palette:

- **Subtle patterns** - Less distracting
- **Bold patterns** - More character
- **Textures** - Natural feel

### Step 3: Adjust Font Size

Pick a size comfortable for your screen:

- **Small screens** - Medium or Large
- **Large screens** - Medium
- **Accessibility** - Large or Extra Large

### Step 4: Fine-tune

Adjust other settings to your preference.

## Tips

### Experiment

Try different combinations! You can always change back.

### Save Favorites

Remember combinations you like. Write them down or take screenshots.

### Match Your Mood

Change themes based on time of day or your mood.

### Consider Lighting

- Bright room → Light themes
- Dark room → Dark themes

### Accessibility

If you have vision difficulties:

- Use high contrast palettes
- Increase font size
- Choose simple backdrops

## Keyboard Shortcuts

| Shortcut | Action              |
| -------- | ------------------- |
| `↑/↓`    | Navigate options    |
| `Enter`  | Apply selection     |
| `Tab`    | Switch sections     |
| `Esc`    | Close Style Manager |

## Troubleshooting

### Theme Looks Wrong

Try a different palette or backdrop combination.

### Text Too Small

Increase font size in Font section.

### Backdrop Not Loading

Wait a moment, complex backdrops take time to render.

### Changes Not Saving

Make sure to click "Apply" button.
