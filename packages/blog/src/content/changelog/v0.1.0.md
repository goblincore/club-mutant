## What's in v0.1.0

This is the first versioned release of Club Mutant. Here's what's included:

### Multiplayer
- Global lobby with real-time player sync
- Custom rooms with password protection and invite links
- Personal "My Room" for each authenticated user

### Characters
- Character select carousel with multiple characters
- Name input and guest mode

### Music
- DJ queue system — take turns playing music
- YouTube search and playback

### In-World OS (OS5000k)
- Windowed mini-OS running inside the 3D world
- DM system, friends list, profile viewer
- Custom wallpapers

### Auth
- Email/password registration and login via Nakama
- Guest mode for quick access
