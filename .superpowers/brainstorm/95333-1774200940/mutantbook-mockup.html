MutantBook Layout

Blue header with profile info, tabbed navigation (Wall / Info / Friends), wall posts with compose box. Classic 2005 Facebook feel inside the OS5000k window.

Key Features

1

Wall Posts

Friends post on your wall, you post on theirs. Chronological feed with avatar, username, message, timestamp.

2

Profile Header

Avatar, bio, vibes count, friend count. Add Friend / Remove Friend button. Navigate to any user's profile.

3

Info Tab

Extended profile: favorite song, links, member since. Uses existing Nakama profile metadata.

4

Friends Tab

Grid of friend avatars with online status. Click to visit their profile. Reuses existing friends.list bridge method.
